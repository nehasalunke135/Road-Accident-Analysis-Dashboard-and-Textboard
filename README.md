# Road-Accident-Analysis-Dashboard-and-Textboard
## Project Overview
<a href="https://github.com/nehasalunke135/Road-Accident-Analysis-Dashboard-and-Textboard/blob/main/Dataset%20with%20Dashboard%20and%20Textboard.xlsx">Project Overview<a>

This project presents a Road Accident Analysis Dashboard and Textboard based on accident data from 2018 to 2023.
The dataset contains 3,000 records, and all records have been fully utilized for analysis. The data used in this project is sourced from Kaggle .
The project includes both a dashboard and a textboard:
The dashboard provides visual insights into accident trends.
The textboard presents key observations and summarized analysis.
This analysis helps in identifying patterns, understanding accident trends over the years, and supporting better decision-making for road safety.
